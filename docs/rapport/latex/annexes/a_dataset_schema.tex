\chapter{Schéma JSON du dataset}
\label{ann:dataset}

Cette annexe regroupe les schémas des fichiers \ac{JSON} qui constituent la partie structurée du dataset de \tamwil{}. Les champs marqués \textit{(optionnel)} peuvent être absents~; les autres sont obligatoires et font l'objet d'une validation au chargement dans \code{data\_loader.py}.
